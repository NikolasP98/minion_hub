    **Derived computations inside the script:**
    - `skillFindings`: findings where `f.chapterId === null`
    - `chapterGroups`: a `Map<string, { name: string; findings: ValidationFinding[] }>` built from findings where `f.chapterId !== null`, keyed by chapterId, using the finding's chapterName for the group header
    - `passingCount`: `skillEditorState.chapters.length - chapterGroups.size` (chapters without any findings)
    - `headerText`: built from validation counts following the copywriting contract:
      - errors only: `"Validation — N error(s)"`
      - errors + warnings: `"Validation — N error(s), N warning(s)"`
      - warnings only: `"Validation — N warning(s)"`
      - all clear: `"Validation — all clear"`
      Pluralize correctly: "1 error" not "1 errors"

    **Template structure (following 07-UI-SPEC.md component contract):**
    ```

      <!-- HEADER: bg2 background, 2.75rem height -->
      <div class="panel-header">
        <span class="panel-title">{headerText}</span>
        <button class="close-btn" onclick={() => { skillEditorState.showValidation = false; skillEditorState.publishAnyway = false; }} aria-label="Close validation panel">
          <X size={14} />
        </button>
      </div>

      <!-- BODY: scrollable -->
      <div class="panel-body">
        <!-- Skill-level findings (if any) -->
        {#if skillFindings.length > 0}
          <div class="chapter-group">
            <div class="chapter-group-header">
              <span class="chapter-group-name">Skill</span>
            </div>
            {#each skillFindings as finding}
              <div class="validation-row {finding.level}">
                {#if finding.level === 'error'}
                  <XCircle size={12} class="finding-icon error" />
                {:else}
                  <AlertTriangle size={12} class="finding-icon warning" />
                {/if}
                <span>{finding.message}</span>
              </div>
            {/each}
          </div>
        {/if}

        <!-- Per-chapter groups -->
        {#each [...chapterGroups.entries()] as [chapterId, group]}
          <div class="chapter-group">
            <div class="chapter-group-header">
              <span class="chapter-group-name">{group.name}</span>
              <button class="fix-btn" onclick={() => {
                const ch = skillEditorState.chapters.find(c => c.id === chapterId);
                if (ch) openConditionOrChapter(ch);
              }}>Fix &#x2192;</button>
            </div>
            {#each group.findings as finding}
              <div class="validation-row {finding.level}">
                {#if finding.level === 'error'}
                  <XCircle size={12} class="finding-icon error" />
                {:else}
                  <AlertTriangle size={12} class="finding-icon warning" />
                {/if}
                <span>{finding.message}</span>
              </div>
            {/each}
          </div>
        {/each}
      </div>

      <!-- FOOTER -->
      <div class="panel-footer">
        <span class="passing-count">{passingCount} chapter{passingCount !== 1 ? 's' : ''} passing</span>
        {#if skillEditorState.publishAnyway}
          <button class="publish-anyway-btn" onclick={() => { publishSkill(); }}>
            Publish Anyway
          </button>
        {/if}
      </div>
    </aside>
    ```
