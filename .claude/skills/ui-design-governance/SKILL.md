# UI Design Governance (Hub mirror)

**Canonical skill: `../.claude/skills/ui-design-governance/SKILL.md`** (meta-repo root, one level above this repo). Read it if reachable ŌĆö it has the full token/primitive tables. The iron rules below stand alone.

## Iron rules

1. **Semantic tokens only** in `.svelte`: no literal hex/rgb, no Tailwind palette utilities (`bg-slate-500`), no arbitrary sizes (`text-[16px]`), no raw shadows/durations/easings, no numeric z-index (`--layer-*` instead). Raw colors are legal only in `app.css` + `themes/*.ts`.
2. **Ōøö Forbidden names (hard-fail):** `--accent`, `--accent-bg`, `--accent-rgb`, `--color-background`, `--color-bg1`, `--color-error`, `--color-primary`, `--color-primary-foreground`. Also: **text tokens never paint surfaces** ŌĆö `--color-muted`/`--color-foreground` alias to TEXT colors; as backgrounds they go gray-on-gray on light themes. Surfaces = `--color-canvas/surface-1..3/overlay` only.
3. **Reuse primitives:** `Button` (never bare `<button>`), themed Select (never native `<select>`), `EmptyState`, `Spinner`/`Skeleton`, `Tooltip`, `FormField`, `StatusDot` (ui copy), `DataTable`, `PageHeader`, Zag wrappers. Hand-rolled copies are ratcheted debt.
4. **Theme = `data-theme` presets only.** Tailwind `dark:` variants are dead code here. Components read tokens; themes swap values.
5. **Never hand-edit `tokens.css`** ŌĆö it is generated from `@minion-stack/design-tokens` `contract.json`.
6. **Layout contracts** (violations shipped 7 bugs, 2026-07-15): one scroll owner per screen ŌĆö content pane owns scroll, section nav never scrolls away; the `(app)/+layout.svelte` fade-wrapper must stay `flex flex-col` (shells below rely on `flex:1` for height); page roots inside `SectionShell` need `flex-1 min-w-0` (or use `PageShell`) ŌĆö bare `h-full` roots shrink to content width and trip EditableGrid into one column; `Button` slots children into an inner `inline-flex` row span consumer classes can't reach ŌĆö override via scoped ancestor `:global(.x > span)` (POS sell `.card`, appearance `.theme-card`); Svelte scoped rules for component children need a real scoped ANCESTOR anchor ŌĆö `.a :global(.b)` never matches a sibling (ShiftBanner `.mini-rail`).
7. **Primitive + panel contracts** (root-caused 2026-07-15, DataTable): forwarded classes on shared primitives are DEAD in plain scoped CSS ŌĆö always `.scoped-ancestor :global(.class)` (+ `> span` for Button's inner row); selection controls = fixed `1rem` border-box both states (`--color-border-strong`/`--color-surface-2` unchecked, `--color-accent`+`--color-on-accent` checked, row-hover promotes border); NEVER dismiss panels via `position:fixed` backdrop (a `backdrop-filter`/`transform` ancestor becomes its containing block and shrinks it ŌĆö DataTable's blurred thead broke click-outside) ŌĆö use `<svelte:document onpointerdown>` + Escape or Zag wrappers.
8. **Bar-row lists** (user directive 2026-07-16): `label | value(s) | bar LAST` ŌĆö numbers left of the bar, bar grows rightward on the final `1fr` track, and bars share one start x via container-owned tracks + `subgrid` rows (per-row grids stagger bars). Reference crm/+page.svelte `.funnel`/`.chmix`.
9. **On-accent prefers white** (user directive 2026-07-16): white text on the accent whenever it clears AA 4.5; black ONLY on bright accents where white can't (amber/cyan/green/orange). Dark accents are tuned to 600-scale shades so white passes ŌĆö never flip a dark accent back to black text; `onAccentFor` (themes/contrast.ts) encodes the rule.
9b. **Severity/status ramp = one fixed semantic map** (user directive 2026-07-16): a status/severity LEVEL maps to ONE status token and reads the SAME hue on EVERY surface ŌĆö chip, chart series, sankey node, dropdown dot, legend. Never let a chip palette drift from the chart palette (shipped: `info` chip `bg-accent`/red vs dropdown `--color-info-fg`/blue). Status tokens only (danger/warning/info/success + neutral/surface for calm steps); **never `--color-accent` for a status level** (accent = primary-action; reads as false alarm). Hub ramp: criticalŌåÆdanger, high/mediumŌåÆwarning, lowŌåÆneutral, infoŌåÆinfo, okŌåÆsuccess.
10. **Selection, charts, sticky** (root-caused 2026-07-16): list/nav SELECTION = accent-tinted surface (`bg-accent/10`) + `--color-accent` text ŌĆö never a full accent fill with on-accent text (that's an action-button style; it drowns icons/dots and goes near-black-on-blue); canvas chart colors (ECharts itemStyle/palettes/sankey nodes) must be RESOLVED strings via `cssVar()`/`chartColors()` from `$lib/utils/chart-colors` ŌĆö raw `'var(--token)'` renders gray and blanks gradient sankeys (DOM inline styles may keep `var()`); `position: sticky` headers need an opaque `--color-surface-*` background ŌĆö transparent variants let content bleed through; every open-able panel has explicit dismissal (outside-pointerdown + Escape, or a Zag wrapper) ŌĆö a `{#if open}` menu with no close path is a bug.
11. **Gates after every UI change:**
   ```bash
   bun run lint:design && bun run lint:tokens
   ```
   Changed-file debt may only DECREASE (CI-enforced). Exceptions: `scripts/design-lint-exceptions.json`, exact file+rule+numeric cap+category+reason. Runtime-authored vars: register in `scripts/token-integrity.mjs` maps.
